Performs tree-shaking analysis to determine which classes are actually used by the user's code, either directly or indirectly through dependencies.

This class analyzes the dependency graph starting from user-defined classes and follows all references to determine the minimal set of classes needed at runtime, similar to how Dart's tree-shaking works.
