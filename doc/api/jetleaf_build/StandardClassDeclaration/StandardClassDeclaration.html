A standard implementation of ClassDeclaration that provides runtime metadata and reflection capabilities for Dart classes.

This class supports inspection of class properties such as fields, methods, constructors, supertypes, and metadata annotations. It can also instantiate class objects reflectively using a provided factory or by matching a suitable constructor.

Example

final reflectedClass = StandardReflectedClass(
  name: 'MyClass',
  type: MyClass,
  parentLibrary: myLibrary,
  constructors: [myConstructor],
  fields: [myField],
  methods: [myMethod],
);

final instance = reflectedClass.newInstance({'value': 42});

In this example, StandardReflectedClass provides access to the metadata and creation mechanism of MyClass, which can be instantiated via newInstance() with a map of named arguments.
