A standard implementation of ParameterDeclaration used to represent metadata about a parameter in a Dart function, method, or constructor.

This class provides information such as the parameter name, type, whether it is optional or named, and whether it has a default value.

Example

final param = StandardReflectedParameter(
  name: 'count',
  type: intType,
  isOptional: true,
  hasDefaultValue: true,
  defaultValue: 5,
);

print(param.getName()); // "count"
print(param.getIsOptional()); // true
print(param.getDefaultValue()); // 5
