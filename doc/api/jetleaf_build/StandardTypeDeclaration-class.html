StandardTypeDeclaration

StandardTypeDeclaration class final

A standard implementation of TypeDeclaration for representing common Dart types, such as primitive types, classes, enums, typedefs, and records.

This class holds metadata such as the type's name, runtime representation, nullability, kind (e.g., class, enum), generic type arguments, and optionally a reference to a SourceDeclaration.

Example

final type = StandardReflectedType(
  name: 'String',
  type: String,
  isNullable: false,
  kind: TypeKind.classType,
  declaration: MyReflectedClassDeclaration(),
);

print(type.getName()); // "String"
print(type.getKind()); // TypeKind.classType
