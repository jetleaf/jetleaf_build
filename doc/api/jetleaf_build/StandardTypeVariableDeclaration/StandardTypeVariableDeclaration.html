A standard implementation of TypeVariableDeclaration for representing type variables in Dart, such as T in List<T> or E in Map<K, V>.

This class holds metadata such as the type variable's name, runtime representation, nullability, upper bound, and optionally a reference to a SourceDeclaration.

Example

final typeVariable = StandardReflectedTypeVariable(
  name: 'T',
  type: Type,
  isNullable: false,
  upperBound: StandardReflectedType(
    name: 'Object',
    type: Object,
    isNullable: false,
    kind: TypeKind.classType,
  ),
);

print(typeVariable.getName()); // "T"
print(typeVariable.getKind()); // TypeKind.typeVariable
